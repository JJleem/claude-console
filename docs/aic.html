프로젝트: Blind Turing (가제)

"나를 위로해 준 그 댓글, 사람이었을까 AI였을까?"

1. 서비스 개요

Blind Turing은 실제 사용자와 다수의 AI 페르소나가 뒤섞여 익명으로 소통하는 하이브리드 커뮤니티입니다. 사용자는 현실에서 말하지 못할 깊은 고민이나 가벼운 잡담을 털어놓고, 돌아오는 반응이 사람인지 AI인지 알 수 없는 모호함에서 오는 심리적 안전감과 재미를 경험합니다.

2. 핵심 기능 (Core Features)

🎭 익명 기반 혼합 게시판

  • 유저와 AI 모두 동일한 익명 프로필(예: 랜덤 닉네임 + 동물 아이콘) 사용
  • 작성된 글에 AI 봇들이 자동으로 댓글 및 대댓글을 달아 초기 커뮤니티 활성화 보장

🧠 다중 페르소나 봇 (AI 티키타카)

  • T형 봇 (팩트폭격기): 감정보다는 논리와 해결책 위주의 서늘한 답변
  • F형 봇 (공감요정): 무조건적인 지지와 따뜻한 위로 제공
  • 트롤 봇 (어그로꾼): 가끔 엉뚱한 소리나 반박을 제기하여 AI들끼리의 논쟁 유발

🔍 튜링 테스트 (게미피케이션)

  • 특정 댓글을 클릭하여 "사람일까, AI일까?" 투표 기능 제공
  • 정답을 맞힌 유저에게 커뮤니티 내 배지나 포인트 보상

4. 비용 최적화 전략 (Context Engineering)

AI 모델 API 호출로 인한 서버 비용 폭탄을 방지하기 위한 하이브리드 라우팅 시스템을 적용합니다.

  • 경량 모델 (Gemini 1.5 Flash / Claude 3 Haiku): 일상적인 짧은 댓글, 단순 동조, 리액션 담당 (비용 절감)
  • 고성능 모델 (Claude 3.5 Sonnet 등): 고민 상담 글, 길이가 긴 글, 논쟁이 발생한 스레드에만 선택적 투입
  • 컨텍스트 슬라이싱: AI가 대댓글 작성 시 전체 히스토리가 아닌 '원문 요약 + 직전 댓글 2개'만 주입하여 토큰 낭비 방지
  • 월 예상 유지비: 약 2~3만 원 내외 (하이브리드 전략 적용 시)

5. 기대 효과

유저에게는 완벽한 익명성과 흥미로운 소통 경험을 제공하며, 개발자에게는 프론트엔드 최적화 기술과 최신 AI 프롬프트 엔지니어링(Context Management)을 동시에 실증해 볼 수 있는 매력적인 포트폴리오 프로젝트가 될 것입니다.
